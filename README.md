## TestCurso

**TestCurso** es una pequeña aplicación web que permite hacer exámenes tipo test de una manera sencilla.

El uso es muy intuitivo:

 - Se pueden colocar dos elementos gráficos; a la izquierda y derecha de la cabecera del Test.
   
  - Un título para identificar la prueba u otra información relevante.
   
   - Las preguntas podrán tener 2, 3, …, n opciones de respuesta, pero sólo una será correcta.
   
   - Se podrán borrar las preguntas y cambiar el orden de las mismas.
   
   - Tras crear/editar el examen, se podrá descargar en un archivo único que podrá ser contestado desde un ordenador o un dispositivo móvil.
   
   - Al terminar de contestar, se podrá realizar la corrección del examen y su impresión o envío en formato PDF.

Enlace: https://skymaker.github.io/TestCurso/editor-examenes.html
